Tipe Tipe Data Di DOM

tipe data keterangan
Document Merupakan Representasi Dari Halman Web
Node Merupakan base class untuk semua tipedata di DOM, seperti, Document, Element,dan Attribut(parent dari seluruh class)
Element merupakan Representasi object element di halaman web, misal head, body, form, input, dan lain lain
NodeList merupakan array/collection yang berisikan Node (kumpulan dari node)
attribute merupakan Representasi dari object attribute dalam element
NamedNodeMap merupakan collection yang berisikan attribute dalam bentuk map(kumpulan dari attribute)
